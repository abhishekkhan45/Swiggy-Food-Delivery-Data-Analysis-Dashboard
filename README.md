# 🍔 Swiggy Food Delivery Data Analysis Dashboard

## 📊 Project Overview

This project is an **Excel-based data analysis and dashboard project** created using Swiggy food delivery data. The objective is to analyze restaurant, food, pricing, rating, and customer review data to identify useful business insights and trends.

The project uses **Excel, PivotTables, PivotCharts, filters, and dashboard visualizations** to transform raw data into an interactive analytical dashboard.

## 📁 Dataset

**File Name:** `Swiggy Raw Data Excel.xlsx`
